
# Операторы

```sql
5 % 4 → 1
2 ^ 3 → 8
2 ^ 3 ^ 3 → 512


-- Square root
|/ 25.0 → 5
||/ 64.0 → 4
||/ 1024 → ?;
power(9, 3) → 729
@ -5.0 → 5.0
```

```sql
SELECT |/ 25; --  оператор квадратного корня (префиксный |/)
sqrt(2) → 1.4142135623730951

SELECT |/ CAST(25 AS double precision);

--abs
SELECT @ '-4.5';
```
